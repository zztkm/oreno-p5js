# おれの p5.js

zztkm の p5.js 作品を保管しておく場所です。

## Requirements

- [Node.js](https://nodejs.org/ja/)

```shell
# install deps
yarn install
```

## 開発サーバー起動

```shell
yarn run dev
```

## Github Pages での公開手順

```shell
yarn run build

yarn run publish
```
